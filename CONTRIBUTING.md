# Contribution Guidelines

Thank you for your interest in contributing to `Kurdish-Logo`! We appreciate all help from the community.

This file contains guidelines to make the process as smooth as possible for both you and us.

## How can you contribute?

*   **Add new logos**: If you are missing a logo for a Kurdish TV channel, you can add it.
*   **Improve existing logos**: You can replace a logo with a higher quality version (e.g., replacing a PNG with an SVG).
*   **Report an issue**: If you find an incorrect, outdated, or low-quality logo, please create an Issue.

## Submitting a change (Pull Request)

Follow these steps to add or change logos:

1.  **Fork the repository**: Create your own copy of the project by clicking "Fork".
2.  **Clone your fork**: `git clone https://github.com/YOUR_USERNAME/Kurdish-Logo.git`
3.  **Create a new branch**: `git checkout -b my-new-logo`
4.  **Add your files**: Add the new logos to the folder. Make sure to follow our guidelines below.
5.  **Commit your changes**: `git commit -m "feat: Add logo for Channel X"`
6.  **Push to your branch**: `git push origin my-new-logo`
7.  **Create a Pull Request**: Go to your forked repository on GitHub and create a new Pull Request. Describe the changes you have made.

## Logo Guidelines

To maintain a high and consistent quality, please follow these rules:

*   **File Format**:
    *   **SVG** is the preferred format. Vectorized logos are scalable and flexible.
    *   If SVG is not available, use a high-resolution **PNG** with a transparent background.

*   **Filename**:
    *   Use lowercase letters and hyphens.
    *   Example: `rudaw-logo.svg`, `kurdsat-news-logo.png`.

*   **Quality**:
    *   The logo should be clean and official.
    *   Avoid including slogans, shadows, or other graphical elements that are not part of the logo itself.

*   **Copyright**:
    *   **Only** upload logos belonging to TV channels.
    *   **Do not** upload promotional images, app screenshots, or other material containing copyrighted images from movies, series, or sports leagues (such as the `rebin4k.png` image which contains logos for the Premier League, Game of Thrones, etc.). This is to avoid legal issues. By submitting a logo, you confirm that you are not violating anyone's copyright.

Thank you for your help in building the best collection of Kurdish channel logos!
